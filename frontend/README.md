<h1>Travel Planner</h1>

A simple web application to plan and manage trips. Users can sign up, log in, add trips with details like hotels, restaurants, transport, tourist attractions, and budget, and view or update their trips.

Technologies Used

Frontend: React.js, HTML, CSS, JavaScript

Backend: Node.js, Express.js

Data Storage: JSON files

Dependencies

Make sure to install dependencies for both backend and frontend:

Backend:

cd backend
npm install

Frontend:

cd frontend
npm install

About trips.json and users.json

These files store your trips and user data locally.

They are ignored in Git for privacy, so they are not pushed to GitHub.

Create them manually in the project folder 

// trips.json
[]
// users.json
[]
